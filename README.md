Техническое задание для приложения "Умный сад"

1. Введение
   
Разработка телеграмм-бота для отслеживания состояния растений.Телеграмм-бот будет предоставлять пользователям возможность мониторинга состоянием своих растений.

2. Цели и задачи
- Создать телеграмм-бота который позволяет пользователям отслеживать состояние растений за счет диаграммы, получать уведомления о необходимости в поливе или уходе.
       
3. Функциональные требования

3.1. Основные функции
- Мониторинг состояния растений при помощи диаграмм
- Возможность добавлять несколько растений в умный сад
- Сортировка расстений по их функциям
  
    
- Уведомления:
  - Настройка уведомлений о необходимости полива.
  
- Рекомендации по уходу:
  - Предоставление рекомендаций по уходу за растениями на основе собранных данных.
