Техническое задание для приложения "Умный сад"

1. Введение
Разработка приложения для отслеживания состояния растений, включая контроль влажности почвы, освещения и температуры. Приложение будет предоставлять пользователям возможность мониторинга и управления состоянием своих растений в реальном времени.

2. Цели и задачи
- Создать приложение для мобильных устройств (iOS и Android), которое позволяет пользователям:
  - Отслеживать уровень влажности почвы.
  - Контролировать уровень освещения.
  - Измерять и отслеживать температуру окружающей среды.
  - Получать уведомления и рекомендации по уходу за растениями.
  
3. Функциональные требования

3.1. Основные функции
  
- Мониторинг состояния растений**:
  - Интеграция с датчиками для измерения:
    - Влажности почвы (в процентах).
    - Уровня освещения (в люксах).
    - Температуры (в градусах Цельсия).
    
- Уведомления:
  - Настройка уведомлений о необходимости полива или изменения условий освещения.
  
- Рекомендации по уходу:
  - Предоставление рекомендаций по уходу за растениями на основе собранных данных.

4. Нефункциональные требования
- Платформы: Приложение должно быть доступно на Android.
- Производительность: Приложение должно  загружаться и реагировать на действия пользователя.
